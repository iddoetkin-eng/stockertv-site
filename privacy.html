Core Privacy Principles

No user-level data

The video layer operates without accessing individual user profiles.

  • No account data needed
  • No behavioral tracking

Platform-level delivery

Content is delivered at the application level, not the individual level.

  • Feed or API-based integration
  • No personal targeting

Minimal processing

Only operational data necessary for delivery and performance.

  • Basic usage metrics
  • Aggregated insights only

Stateless architecture

Built with modern enterprise privacy expectations in mind.

  • Clear data boundaries
  • No cross-session tracking
